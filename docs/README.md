## 项目核心信息

### 项目名称
AI Trading Mate - AI驱动交易助手平台

### 项目目标
为投资者提供实时股票分析、技术指标、K线图表和AI生成的投资见解

### 核心技术栈
- **前端**: React + TypeScript + Tailwind CSS + ECharts
- **后端**: FastAPI + Python + SQLAlchemy
- **数据库**: PostgreSQL + Redis
- **认证**: Auth0
- **AI**: DeepSeek V3.2, Kimi K2

### 主要功能模块
1. 用户认证与账户管理
2. 市场数据获取与展示
3. K线图表与技术指标
4. AI分析与建议
5. 自选股管理
6. 交易脚本管理
7. 系统管理
8. 数据存储与历史
