# Project Summary
AI Trading Mate is an advanced AI-driven trading assistant platform that optimizes investment strategies through real-time stock analysis tools. It integrates market data from sources like Tushare and AKShare, providing features such as K-line charts, technical indicators, and AI-generated insights. The platform enhances trading decision-making and data visualization, delivering a superior user experience. Recent updates focused on refining AI chat functionalities and improving backend configurations for better performance.

# Project Module Description
- **Frontend**: Built with React, TypeScript, and Tailwind CSS, this module offers an intuitive user interface for stock analysis and trading. Recent updates enhance chat interactions and user experience.
- **Backend**: Developed using FastAPI, this module provides a robust API for data retrieval from Tushare and AKShare, utilizing Redis for efficient caching. Updates include improved database configuration management.
- **AI Analysis**: Implements AI models to generate insights and trading scripts from stock data, allowing real-time analysis and customizable settings.
- **Authentication**: Managed by Auth0, it supports user login via GitHub and Google accounts.
- **Stock Search**: Enhanced to support improved algorithms and caching for faster responses.
- **Script Monitoring**: Enables users to generate and monitor trading scripts based on AI analysis.
- **User Settings**: Configures AI API settings, including model selection and usage limits for free-tier users.
- **Cache Management**: Implements backend caching for efficient data retrieval.
- **Data Source Management**: Manages multiple data sources with prioritized selection for seamless data retrieval.
- **Trading State Management**: Maintains user context during trading sessions.

# File Description Inventory
- **backend/**: Contains all backend-related code, including API routes, services, models, and database migrations.
- **frontend/**: Contains all frontend-related code, including React components, pages, contexts, and styles.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS, ECharts, Framer Motion
- **Backend**: FastAPI, Python, Tushare API, AKShare API, Redis
- **AI**: DeepSeek V3.2 and Kimi K2 for AI-driven analysis
- **Authentication**: Auth0 for user management
- **Database**: PostgreSQL (Neon) for production data storage

# Usage
1. **Install Dependencies**:
   - For the frontend:
     ```
     pnpm install
     ```
   - For backend dependencies:
     ```
     pip install -r requirements.txt
     ```

2. **Build the Project**:
   - For the frontend:
     ```
     pnpm run build
     ```

3. **Run the Backend**:
   - Start the FastAPI application:
     ```
     uvicorn main:app --reload
     ```

4. **Run the Frontend**:
   - Start the frontend application:
     ```
     pnpm run start
     ```
